# 🚀 Addoners Optimizer

Smart FPS Optimization for Minecraft (Fabric)

Addoners Optimizer is a dynamic performance engine that automatically adjusts your game settings based on real-time FPS to deliver smoother gameplay—no manual tweaking required.

---

⚡ Features

- 🎯 Automatic FPS Optimization
  Dynamically adjusts settings based on performance

- 🧠 Adaptive Optimization Engine
  Responds to FPS changes in real-time

- 🎮 Works with ALL Shaders
  Compatible with any shader pack

- 🔥 Enhanced for Addoners Shaders
  Extra performance boost with Quartzglow & Skygleam

- 📄 .addoners Profile System
  Custom optimization profiles for different setups

---

📦 Installation

1. Install Fabric Loader
2. Install Fabric API
3. Place the mod ".jar" in your "mods" folder
4. Launch Minecraft

---

⚙️ Usage

Just install and play.

The optimizer automatically:

- monitors FPS
- adjusts settings
- improves performance

No configuration required.

---

⚡ Pro Tip

For maximum performance, use with:

- Quartzglow Shaders
- Skygleam Shaders

---

🧪 Compatibility

- ✅ Minecraft (Fabric)
- ✅ All shader packs
- ✅ Low-end & mid-end PCs

---

📂 Profiles

Profiles are stored in:

config/teamaddoners/profiles/

Example:

{
  "type": "optimizer_profile",
  "settings": {
    "renderDistance": 8
  }
}
